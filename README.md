# ⚡ CatchUp

> Never let a pending deal slip through the cracks. CatchUp is your smart financial assistant that tracks pending invoices, automates follow-up emails, and lets clients pay instantly.

**🌐 Live Demo:** [https://catchup-x6sy.onrender.com/](https://catchup-x6sy.onrender.com/)

---

## 🚀 Key Features

* **Automated Drip Emails**  
  Set your follow-up delay, customize your templates, and let CatchUp chase down unpaid invoices while you sleep.
  
* **1-Click Stripe Payments**  
  Generate custom, secure Stripe payment links directly from your dashboard and drop them into your emails.

* **AI Revenue Forecasting**  
  Know exactly how much money is sitting in your pipeline and get AI-driven predictions on expected monthly payouts.

* **High-Converting Marketing Page**  
  A stunning, modern landing page featuring realistic CSS mockups, dynamic gradients, and unified branding designed to attract freelancers and agencies.

* **Dynamic SaaS Dashboard**  
  Track leads, tasks, and reports in a beautiful interface. Includes real-time status tracking (Pending, Contacted, Recovered).

* **Flawless Light/Dark Mode**  
  A premium UI that instantly toggles between a clean white surface and a sleek slate-blue dark theme.

## 🛠 Tech Stack

* **Frontend**: React.js, Vite, Lucide-React
* **Backend**: Node.js, Express
* **Payments Integration**: Stripe

## 💻 Getting Started

1. **Clone the repository**
2. **Install frontend dependencies**: 
   ```bash
   cd catchup-app
   npm install
   ```
3. **Install backend dependencies**:
   ```bash
   cd catchup-backend
   npm install
   ```
4. **Run the backend**:
   ```bash
   cd catchup-backend
   npm run start
   ```
5. **Run the frontend**:
   ```bash
   cd catchup-app
   npm run dev
   ```

## 🎨 UI/UX Highlights

* **Unified Branding**: Uses the sleek `Outfit` Google font globally.
* **Responsive Design**: Adapts beautifully across desktops, tablets, and phones, including a mobile-native drill-down menu for settings view.
* **Click-Outside Dismissals**: Interactive elements like notification lists and modal popups automatically close when clicking anywhere outside of them.
* **Micro-interactions**: Smooth hover states, float animations on mockups, hover close buttons, and sliding toast notifications.
* **Ultra-Mobile Optimization**: Action buttons automatically morph into space-saving circular icons on screens under 500px to maintain clean alignment.
